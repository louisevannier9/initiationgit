Qu'est-ce que le web ?

Création ARPANET en 1969 ( ancêtre d'internet militaire et universitaire) Apparition des e-mail 1972 Création du web 1991 par Tim Berners-Lee ( Pages et hyperliens ) Création du W3C 1994 ( Standardisation du Web ) WWW : WORLD WIDE WEB HTTPS : HYPERTEXT TRANSFER PROTOCOL SECURE W3C se divise en plusieurs facteurs HTML / CMS / HTTPS Internet : Infrastructure mondiale Web : service sur internet Cloud et service en ligne PaaS : Platform as a service IaaS : infrastructure as a Service SaaS : software as a service Avantage : scalabilité, accessible partout, maintenance délégué Permet le stockage de données ( Google Drive, Dropbox) Exécution d'applications (Gmail, Office 365) Et l'hébergement de sites web Cycle Client-Serveur Request -> client -(HTTP Request )> interpretation -(Display)> Rendered WebPage Le navigateur Rôle et moteur de rendu Interface Client-Serveur Il sert d'interface entre l'utilisateur et le serveur web. Il permet d'accéder recevoir et afficher les pages web selon un modèle client/serveur. Langages interpréter cote client: - HTML - CSS - JAVA Moteurs de rendu principaux : BLINK -> Chrome, Edge Gecko ->Firefox Webkit -> Safari Outil pour les Développeurs: DevTools Backend : Serveur 2 grandes familles . Bases SQL (relationnelles) Ex : MySQL, PostgreSQL, SQL Server. . Bases NoSQL (non relationnelles ) Ex : MongoDB, Cassandra, Redis CSR : Client Side Rendering SSR : Server Side Rendering
